Política de Privacidade – WhatsTime Última atualização: 2025 O aplicativo WhatsTime foi desenvolvido com o objetivo de ajudar o usuário a agendar o envio de mensagens através do WhatsApp de forma organizada e prática. Levamos a sua privacidade a sério. Esta política explica, de forma simples, quais dados são utilizados e com qual finalidade. 1. Dados coletados O WhatsTime não coleta, armazena ou vende dados pessoais em servidores próprios. Todas as informações de mensagens agendadas são armazenadas localmente no dispositivo do usuário, em um banco de dados interno do aplicativo. Os dados utilizados incluem: Número de telefone informado pelo usuário para envio da mensagem Conteúdo da mensagem cadastrada Data e hora de agendamento Eventualmente, nomes e números vindos da agenda de contatos do dispositivo (apenas para facilitar o preenchimento dos campos dentro do app) Esses dados são usados exclusivamente para realizar o agendamento e envio da mensagem. 2. Permissões utilizadas O WhatsTime pode solicitar as seguintes permissões: Leitura de contatos (READ_CONTACTS) Utilizada apenas para listar e selecionar contatos diretamente da agenda do dispositivo, facilitando o preenchimento do número de telefone. O aplicativo não copia, envia ou armazena esses contatos em servidores externos. Acesso à internet (INTERNET) Utilizado para se comunicar com um servidor backend próprio do WhatsTime, que por sua vez integra com a WhatsApp Cloud API (serviço oficial da Meta) para realizar o envio das mensagens. 3. Integração com WhatsApp Cloud API O WhatsTime utiliza a WhatsApp Cloud API, serviço oficial fornecido pela Meta Platforms, Inc., para efetuar o envio das mensagens agendadas. O envio é feito através de um número de WhatsApp Business configurado pelo desenvolvedor ou pela empresa usuária. O aplicativo não tem acesso ao conteúdo das conversas completas do usuário dentro do WhatsApp, apenas à mensagem que o próprio usuário agenda para envio. O uso da WhatsApp Cloud API está sujeito também aos Termos de Uso e Política de Privacidade da Meta. 4. Armazenamento e segurança As mensagens agendadas ficam armazenadas localmente no dispositivo em banco de dados interno do app. Dados enviados ao servidor backend (como número e mensagem) são transmitidos utilizando HTTPS sempre que possível. Não são armazenadas senhas de contas de WhatsApp nem dados de login do usuário. 5. Compartilhamento de dados O WhatsTime não vende, aluga ou compartilha dados pessoais com terceiros para fins de marketing. As informações podem ser compartilhadas apenas: com provedores de infraestrutura estritamente necessários à prestação do serviço (por exemplo, servidores de hospedagem do backend); quando exigido por lei ou ordem judicial. 6. Direitos do usuário O usuário pode: Desinstalar o aplicativo a qualquer momento, removendo os dados locais armazenados. Entrar em contato com o desenvolvedor para tirar dúvidas ou solicitar informações adicionais sobre privacidade e dados. 7.
